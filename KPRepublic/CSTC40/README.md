Crazy mad tap-dance layout for the CSTC40

Double-click:

```
1! ESC
qQ TAB
`~ \|
[{ -_
]} =+
'" UP
,< LEFT
.> DOWN
/? RIGHT
4$ control-shift-command-4 (it's a Mac thing)
```

Hold:

```
aA SHIFT
zZ CONTROL
xX COMMAND
cC OPTION
vV MO(1)
bB MO(2)
nN MO(3)
mM MO(4)
ENTER SHIFT
```

Layer shifts on `azxc` get you shift-A control-Z command-X option-C. Other layer shifts are obvious but unnecessary.
